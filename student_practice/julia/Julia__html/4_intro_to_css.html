Intro to CSS

CSS (Cascading Style Sheets) is a language used to define the visual presentation of HTML and XML documents on the web. It enables designers to specify how elements should appear, such as their colors, fonts, spacing, and layout.

CSS selectors are patterns that target specific elements within a webpage, allowing designers to apply styles selectively. For example, the dot notation (.) in CSS is used to target elements with a specific 'class'.

For instance, .day-report would apply styles to all elements that have class="day-report" in their HTML, allowing consistent styling across multiple elements with the same class. This notation is essential for organizing and applying styles efficiently across different parts of a website.

Intro to HTML

This is a website!

HTML is made out of HTML 'elements'. An element in HTML is a fundamental building block used to structure and organize content on a webpage. It acts like a container or a box that can hold other HTML elements like text, images, or other divs. By applying CSS styles to divs, you can control their appearance, layout, and positioning on the page, making them versatile for creating columns, sections, or entire layouts.

Elements are pieces of content that are wrapped in HTML 'tags'. These tags are structured as pairs that 'open' and 'close'. For example,

<div> Hello World! </div>

is a 'div' element that contains the text, 'Hello World'.

Each tag opens and closes with the matching element type (in chis case, 'div'), and the slash (/) on the second tag indicates that we're closing the tag.

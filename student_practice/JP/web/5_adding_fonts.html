Adding Google Fonts

Challenge: Try picking and choosing a font from Google Fonts to this page.

You can follow this tutorial, with a small modification. Google Fonts will now offer you pre-created classes that look like this: 

.roboto-bold {
  font-family: "Roboto", sans-serif;
  font-weight: 700;
  font-style: normal;
}
You can copy-paste only the css attributes (starting with 'font-') into the CSS class of your choice. For example: 
.action-card .title {
  font-size: 1.2em;
  font-family: "Roboto", sans-serif;
  font-weight: 700;
  font-style: normal;
}
Remember that this will only work after you've also copy-pasted the Google Fonts embed code!


.edu-au-vic-wa-nt-guides-unique {
  font-family: "Edu AU VIC WA NT Guides", cursive;
  font-optical-sizing: auto;
  font-weight: 400; /* You can use numeric values like 400 or 700, or 'bold', 'normal' */
  font-style: normal;
}
